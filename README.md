#### Code Snippet: FULL OUTER JOIN

The below code snippet demonstrating a **FULL OUTER JOIN** between two tables.

```sql
SELECT column_name(s)
FROM table1
FULL OUTER JOIN table2
ON table1.column_name = table2.column_name
WHERE condition;
```
